Список команд для CRUD над файлами

  1. touch — команда Unix, призначена для встановлення часу останньої зміни файлу або доступу в поточний час.

  2. cat — утиліта UNIX-подібних систем, яка здійснює операцію конкатенації вмісту вхідних аргументів та виводить їх на стандартний вивід.

  3. echo (англ. Echo — луна) — команда в DOS, OS/2, Microsoft Windows, Unix і Unix-подібних операційних системах, яка виводить рядок тексту на комп'ютерний термінал. Це зазвичай використовується в сценаріях оболонок і командних файлах для виводу тексту про стан скрипту на екран або у файл.

  4. rm (від англ. remove) — утиліта у UNIX та UNIX-подібних системах, що використовується для видалення файлів з файлової системи.
